Your ETP Project Home Page

This is the index.html page for your deployed Elastic Transaction Platform project. Change this page or go directly to the first screen of your ETP transaction with the URL:

The index.html and other static content such as the cascading style sheets that control the look and feel of your screens are in the Java EE folder of your ETP Deployment Project. You can change the ETP region name part of the URL (for Java EE deployments) and the /servlet portion of the URL in the ETP Deploy Settings file. For servlet container deployments the URL is determined by the name of the war you deployed.

Control will transfer to this region's INIT transaction screen in seconds.
